Фітнес-трекер (лайт) — запис пробіжок або тренувань (кілометри, повтори).
1)
Користувач матиме можливість переглядати кількість пройдених кроків за день, створювати тренування, а програма показуватиме, скільки ще потрібно пройти, враховуючи вже зроблені кроки та інші дії, що впливають на витрату калорій (за наявності відповідного браслета або годинника)..
Також вона відображатиме кількість втрачених калорій завдяки вимірюванню пульсу ( також за наявності відповідного браслета або годинника).
Користувач зможе переглядати вправи, техніку їх виконання та темп — відповідно до критеріїв, які він вкаже (вага, зріст та цілі).

Як користувач, я хочу можливість ділитись результатами щоб бачити результати друзів
Як користувач, я хочу версію цього додатку на годинник щоб було зручніше користуватись під час тренувань
Як користувач, я хочу бачити аналіз якості сну, щоб розуміти, як відпочинок впливає на мої результати тренувань.
Як користувач, я хочу брати участь у щоденних викликах та челенджах, щоб залишатися мотивованим.
Як користувач, я хочу користуватися голосовим асистентом під час тренувань, щоб не відволікатися на телефон і зосереджуватися на вправах.

Функція ShareResults(користувач, результат):
якщо результат = null:
повернути "Помилка: немає даних для публікації"
додати результат у профіль користувача
сповістити друзів користувача про новий результат
повернути "Результат успішно опубліковано"

Функція ViewFriendsResults(користувач):
отримати список друзів користувача
для кожного друга:
отримати останні опубліковані результати
відобразити стрічку результатів друзів

5)
Публікація результату користувача
Користувач ділиться своїм результатом
Результат з’являється у стрічці друзів і в профілі користувача.

Публікація без результату
Користувач натискає “Поділитися”, але дані про результат відсутні.
Показується повідомлення “Помилка: немає даних для публікації”.

Перегляд результатів друзів
Користувач має друзів, які поділилися своїми результатами.
У стрічці з’являються пости з даними друзів

Негативний варіант:
У користувача немає друзів або ніхто не публікував результатів.
Виводиться повідомлення “Поки немає результатів для відображення”.

Додавання нового друга
Користувач додає нового друга, який має опубліковані результати.
Після оновлення сторінки з’являються його досягнення.

Користувач додає друга, але той ще не ділиться жодними результатами.
Стрічка не змінюється або показує повідомлення “Нових результатів немає”.
